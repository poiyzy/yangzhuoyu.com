Git颜色高亮

  • Feb 20, 2012

在 Terminal 中使用 git status 和 git diff 时，没有高亮，非常费眼，效率低。

可以在 git 中设置颜色高亮来解决这个问题。输入如下命令：

$ git config --global color.ui true
